# Longbridge People Justin V2 Showcase

Static review site for the Longbridge Justin V2 article drafts.

## Contents

- `index.html` - shareable website with all three rendered articles and images
- `01-reddit-ai-market-data-justin-v2.md` - Reddit draft source
- `02-xueqiu-ai-screening-justin-v2.md` - Xueqiu draft source
- `03-x-earnings-workflow-thread-justin-v2.md` - X / Twitter draft source
- `graphics/` - generated cover and infographic images
- `build-site.mjs` - local generator for rebuilding `index.html`

## Rebuild

```bash
node build-site.mjs
```
